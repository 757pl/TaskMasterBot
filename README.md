# School Task Bot

Telegram бот для управления учебными задачами и расписанием.

## Команды
- /start - начать работу
- /help - помощь
- /myid - узнать свой ID

## Функции
- 📚 Расписание 10Г
- 📝 Управление задачами
- ⏰ Напоминания в 7:00
- 👑 Админ-панель

## Установка
1. Клонировать репозиторий
2. Создать файл `.env` с токеном
3. Установить зависимости: `pip install -r requirements.txt`
4. Запустить: `python bot.py`
